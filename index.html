About This Project

This interactive 3D transformation studio is built with Three.js, an open-source JavaScript library for creating 3D graphics in the browser using WebGL. Explore different geometries and apply various transformations including rotation, scaling, and translation. The scene features dynamic lighting with multiple colored point lights to enhance the 3D effect.
